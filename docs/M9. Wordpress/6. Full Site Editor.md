A criação de um site baseado no editor de blocos Gutenberg, torna-se cada vez mais viável a partir da versão 6.7 do WordPress. A maturidade do editor já se equipara aos de alguns editores famosos, como o [Elementor](https://pt.wordpress.org/plugins/elementor/)
Vamos pegar num tema base para a realização de um site, vamos usar à data o tema [Twenty-Five](https://pt.wordpress.org/themes/twentytwentyfive/)
# Design 

Na área de design do template podemos verificar e adaptar a tipografia, a palete de cores e os blocos base do site. Também é aqui que podemos colocar as classes CSS transversais a todo o site.

# Criar ou adaptar blocos

Na secção de Design do tema também temos acesso à modificação dos blocos de conteúdo existentes. 
Por exemplo, altera o aspeto dos botões, inclusive das suas variações.

![[Pasted image 20260114224548.png|300]]

# Header e Footer

O header e o footer são elementos de um site que se repetem em todas ou na maioria das páginas. Assim deverão fazer parte do template associado às paginas do site. 

![[Pasted image 20260114232609.png]]


# Páginas

Criar cada página do site, que por defeito ficam associadas ao templates `Pages`. 
Para realizar o layout da página com base no design criado é importante organizar os blocos de conteúdo em grupos utilizando os blocos de design, como Columns, Row e Stack.

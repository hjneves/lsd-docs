

GSAP (GreenSock Animation Platform) é considerado o padrão da indústria para animações web profissionais:

- ⚡ **Performance superior** - 20x mais rápido que CSS animations
- 🎯 **Controlo preciso** - Frame-perfect animations
- 🔧 **ScrollTrigger poderoso** - Animações baseadas em scroll avançadas
- 📱 **Cross-browser** - Funciona em todos os browsers
- 🎨 **Flexibilidade total** - Anima qualquer propriedade CSS

O [GSAP](https://gsap.com/) é uma ferramenta de código pensada para criar animações para a web, desde as mais simples e lineares até às mais complexas e com dependências. 
Com recurso a plugins consegue diversos efeitos de animação e diferentes tipos de triggers, como por exemplo os baseados no scroll da página.

# Instalação

Para testar o GSAP podemos criar um codepen, adicionar o package GSAP e fica logo disponível para ser usado. Podes começar com este [GSAP Starter](https://codepen.io/editor/hjneves/pen/019d1bca-a313-71e5-a29e-bbf6a70f7b73)

Sendo uma framework agnóstica pode ser instalada de diversas formas conforme a tecnologia web usada. No nosso caso vamos usar  a versão para desenvolvimento web nativo e por isso basta indicar apenas o script disponibilizado para o efeito. Deverá ser colocado **antes** dos nossos scripts que vão usar o GSAP

```html
<html>
  <head>
     ...
  </head>

  <body>
    <div className="animateMe"></div>
    <h1>Hello GSAP!</h1>

    <!-- Put your script after the HTML -->
    <script src="https://cdn.jsdelivr.net/npm/gsap@3.14.2/dist/gsap.min.js"></script>
    <!-- And put your own code after - any GSAP code you write needs to have access to the library -->
    <script>
    // custom code using GSAP
    </script>
  </body>
</html>
```

Se necessitarmos de usar plugins será necessário adicionar outros links tal como indicado [aqui](https://gsap.com/docs/v3/Installation?tab=cdn&module=esm&require=false#install-helper)

# Easing

O easing de uma animação é determinante para uma correta execução da mesma. O easing determinada forma como as propriedades são a alteradas nos elementos no durante o tempo da animação.
Para além o easing linear e do ease in/out o GSAP tem diversas possibilidades para a curva do easing, sendo a curva `power1.in` o valor por defeito. Nesta [página](https://gsap.com/docs/v3/Eases/) podemos visualizar os diversos tipos de easing.

A escolha acertada do easing para uma determinada animação pode ser suficiente para que o resultado final seja o desejado.

De seguida são apresentados alguns tipos de easing menos habituais com exemplos práticos de utilização

## Menu Lateral (ease back)

```html
<button onclick="openMenu()">Open</button>
<nav class="position-absolute top-0 vh-100 bg-white">
  <button onclick="closeMenu()">Close</button>
  <p class="fs-1 px-3">Hello</p>
  <p class="fs-1 px-3">Projetos</p>
  <P class="fs-1 px-3">Serviços</P>
</nav>

<style>
body {
  background-color: black;
}

nav {
  padding-left: 30px;
  width: 300px;
  transform: translateX(-300px);
}
</style>

<script>
const openMenu = () => gsap.to("nav", { duration: 0.5, ease: "back", x: -30 })
const closeMenu = () => gsap.to("nav", { duration: 0.5, ease: "back.in", x: -300 })
</script>
```

# Origem das transformações

A propriedade `transformOrigin`permite localizar o centro das transformação no elemento. Por defeito situa-se nos 50% 50% (x e y) mas dependendo do tipo de animação a realizar podemos alterar para outro ponto.

Neste exemplo temos uma animação de reveal através da rotação de um elemento quadrado mas usando como origem de rotação o ponto 0% 100%, que significa o canto inferior esquerdo.

```html
<section class="d-flex justify-content-center align-items-center flex-column text-white vh-100 overflow-hidden">
  <p class="display-1 fw-bold px-3">Welcome</p>
  <div class="curtain position-absolute w-100 vh-100 top-0 start-0 bg-danger"></div>
</section>
<section class="d-flex justify-content-center align-items-center flex-column text-white vh-100 overflow-hidden">
  <p class="display-1 fw-bold px-3">Back</p>
</section>

<style>

body {
  background-color: black;
}

.curtain {
  transform: scaleY(1.3)
}
</style>

<script>
gsap.to(".curtain", { 
  delay:0.5, 
  rotate: 90, 
  transformOrigin: "0% 100%", 
  duration: 2, 
  ease: "power4.out", 
  onComplete: () => gsap.set(".curtain", {display: "none"})} )spo
</script>
```

# Stagger

A propriedade `stagger`permite que um tween quando aplicada a vários elementos html desencadeie a animação com um atraso em cada um deles. Por exemplo, se definirmos um valor de `stagger: 0.2`significa que o segundo elemento arranca passados 200ms, o terceiro, passado 400ms e assim sucessivamente.

# Controlo

Uma animação GSAP pode ser criada e controlada em termos da sua execução. Podemos colocar a animação numa variável e depois executar `play(), stop(), pause()` e `reverse()`.
Por exemplo na animação do menu lateral podemos criar uma versão melhorada usando estes controlos. Nota sobre a propriedade `paused`que garante que a animação não é imediatamente executada.

```js

let tween = gsap.to("nav", { duration: 0.5, ease: "back", x: -30, paused: true })

const openMenu = () => tween.play()
const closeMenu = () => tween.reverse()

```

## Exemplo Reveal word by word

Split das palavras do text num array para ser aplicado ao GSAP através da class `char`.

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js"></script>

    <style>
      body {
        font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
      }

      .text-section {
        min-height: 100vh;
        display: flex;
        flex-direction: column;
        justify-content: center;
        align-items: center;
        padding: 50px;
        text-align: center;
        background: #f8f9fa;
      }

      .split-text {
        font-size: 3rem;
        font-weight: bold;
        margin-bottom: 2rem;
        color: #333;
      }

      .char {
        display: inline-block;
        opacity: 0;
        transform: translateY(100px);
      }
    </style>
  </head>

  <section class="text-section">
    <h2 class="split-text">Um relógio para a sua saúde</h2>
  </section>

  <script>
    gsap.registerPlugin(ScrollTrigger);

    // 1. Split text em caracteres
    function splitTextIntoChars(selector) {
      const element = document.querySelector(selector);
      const text = element.textContent;
      element.innerHTML = "";

      text.split(" ").forEach((char) => {
        const span = document.createElement("span");
        span.textContent = char + "\u00A0";
        span.className = "char";
        element.appendChild(span);
      });
    }

    // 2. Aplicar split
    splitTextIntoChars(".split-text");

    // 3. Animar caracteres
    gsap.to(".char", {
      y: 0,
      opacity: 1,
      duration: 0.8,
      stagger: 0.2,
      ease: "power2.out",
      scrollTrigger: {
        trigger: ".split-text",
        start: "top 80%",
      },
    });
  </script>
</html>
```

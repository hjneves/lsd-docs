# Acerca
Hoje em dia já não se desenvolve web a partir do zero. Ou seja, não se cria CSS de raiz, mas sim escolhe-se uma framework para servir de boilerplate.

****
O [Bootstrap](https://getbootstrap.com) é uma das frameworks que tem sido muito usada no desenvolvimento de web sites. È baseado em componentes, tem uma abordagem mobile first e foi inicialmente desenvolvido pela equipa do Twitter.
Os componentes por sua vez são formatados através de um conjunto estruturado de classes CSS que a framework disponibilizada para utilizarmos. Os [docs](https://getbootstrap.com/docs/5.3/getting-started/introduction/) são o recurso indicado para esclarecer tudos sobre as classes disponíveis. Por exemplo a class [container](https://getbootstrap.com/docs/5.3/layout/containers/)
Permite de uma forma rápida realizar o layout de um site, com componentes e formatações pré-definidas. Deixou de requerer JQuery a partir da versão 5.0 sendo atualmente  compatível com Vue e React compatible.

# Instalação
Vamos usar a versão 5 (atualmente temos a versão 5.3.x)
A página de download encontra-se [aqui](https://getbootstrap.com/docs/5.3/getting-started/download/) 

A instalação de uma framework e livraria  CSS consiste em integrar normalmente duas componentes:
- componente CSS, que contém a definição de todas as classes dos componentes e utilitárias
- componente JS, que contém a componente interativa da framework.

Existem normalmente duas abordagens para a instalação do Bootstrap, e o mesmo se aplica genericamente a outras livrarias CSS.
## Local
Numa instalação local os ficheiros da framework são descarregados e integrados na diretoria do projeto.

## Remota
Talvez a forma mais rápida de começar a usar o bootstrap seja através da inclusão nos ficheiros html do projeto, dos links para as componentes do bootstrap disponibilizadas por um CDN
Ver [aqui](https://getbootstrap.com/docs/5.3/getting-started/download/#cdn-via-jsdelivr)


## O que são modelos de IA

Os modelos de IA generalizados, também designados de LLM's (Large Language Models) são modelos de aprendizagem de máquina, em concreto redes neuronais, que foram treinadas em quantidades vastíssimas de dados textuais com o objetivo de encontrar padrões para prompts escritos pelos utilizadores. Na prática estes modelos tentam encontram o melhor padrão de resposta para cada questão que lhes é colocada.

Quando colocamos um pedaço de código HTML num modelo para ele analisar, na prática o que o modelo vai fazer é encontrar através da sua aprendizagem qual o padrão de resposta que faz melhor match com o código questionado. Daqui se consegue perceber a dimensão que estes modelos têm para satisfazerem estes pedidos de prompt.

Qual a tua experiência com a IA? Também conhecido por vibe coding, será certamente uma tendência crescente ao longo tempo, mas para a qual é necessário uma análise e pensamento crítico na sua utilização.

## Boas práticas

Os modelos LLM de IA foram pensados para serem utilizados em contextos muito diversos, dando-nos a sensação que sabem de tudo. A realidade mostra que em situações muito específicas podem divagar ou até mesmo cometer erros, entrando num processo de alucinação.

Uma estratégia de utilização adequada e com uma visão crítica por parte dos utilizadores é onde a estratégia IDI encaixa. Encontrar use cases onde realmente a IA pode ter um contributo positivo (imagine) ter sempre uma visão crítica sobre os resultados obtidos, consultando as fontes utilizadas, se for o caso (discern) e por fim integrar os resultados no contexto de trabalho (integrate).

## Prompts

A engenharia dos prompts determina em muito a qualidade das respostas obtidas. Um bom prompt deverá ter as seguintes características:
- ser específico e detalhado
- incluir contexto relevante
- especificar requisitos técnicos
- pedir explicações quando necessário

Exemplo extremos de dois prompts, um fraco e um mais interessante

*Prompt fraco:*
"Cria uma página web"

*Prompt forte:*
"Cria uma página HTML com CSS incorporado para um portfólio pessoal. Deve incluir: cabeçalho com navegação, secção hero com nome e título profissional, secção de projetos com 3 cartões numa grelha responsive, rodapé com links para redes sociais. Usa cores modernas e design minimalista. Adiciona comentários no código."

## Criação de uma landing page simples

Vamos neste exemplo prático pedir a um modelo de IA para criar um *landing page* simples com um determinado layout e um conjunto de secções baseados num estilo.

### Exemplo de prompt
Cria uma landing page para um negócio de costura usando bootstrap com as seguintes secções: uma imagem de fundo na página, um menu, um breve parágrafo em destaque e depois 3 secções a apresentarem o negócio, uma sobre trabalhos, outra sobre os tipos de serviços e uma final com um form de contacto. No footer inclui alguns icones com link para as redes sociais. Usa um design elegante e uma tipografia com serifa. 


## Geração de código avançado

No prompt iniciar com o contexto em termos de tecnologias a utilizar para que as respostas aos pedidos sejam mais coerentes com o que estamos a trabalhar.
Por exemplo, pedir para criar um form de signup. Indicar que será usada numa determinada framework, se vai ter ou não validação dos campos.

*Por exemplo:*
"Cria um form usando a framework Bootstrap para realizar login do utilizador. Efectua validação dos campos e inclui mensagens de erro no caso de login inválido".
